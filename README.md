# Angular-CLI

## Installation d'Angular

```cmd
npm init
npm install
npm install -g @angular/cli
```

## Creation d'une nouvelle app

```cmd
ng new [app]
```

On peux directement configurer en ajoutant les argument:

```cmd
ng new [app] --routing --prefix cs
```


## Executer un projet

```cmd
cd [app]
ng serve
```
